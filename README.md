# CNCF Study

CNCF(Cloud Native Computing Foundation) 프로젝트들을 학습하고 정리하기 위한 레포지토리.

## 목적

- CNCF Landscape에 포함된 프로젝트들의 역할과 관계를 이해한다
- 주요 프로젝트를 직접 실습하고 기록을 남긴다
- 클라우드 네이티브 생태계 전반에 대한 지식을 체계적으로 쌓는다
